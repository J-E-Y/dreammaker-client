# dreammaker-client


# 소개 

> 어린 친구들이 성적이나 환경에 따라 자신만의 개성없이 진로를 정하게 되는 것을 막기 위해 심리 검사 등을 통해서 어린 친구들에게 각자에게 맞는 꿈을 찾아주는 프로젝트

# 기술 스택

* Front End	React,templated
* Back End	express,mysql,nodejs,sequelize
* Etc	api,eslint,prettier,redux
